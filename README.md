# 📖 Bookstore <a name="about-project"></a>

- Bookstore is a web application where users can display a list of books, add book, and remove a selected book.

## 🛠 Built With <a name="HTML, CSS, JS and REACT"></a>

### Tech Stack <a name="Front end (REACT)"></a>

- This is a mobile and desktop browser responsive website.

  Client
  
- HTML, CSS, JS, REACT
- Technologies Used- HTML, CSS, JS, REACT, GIT, Linters (GitHub Actions, Lighthouse, Webhint, Stylelint, Eslint)

### Key Features <a name="key-features"></a>

- Display a list of books
- Add a book
- Remove a selected book

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Git
- An IDE (VsCode, Atom, etc)

### Setup

Clone this repository to your desired folder:

- git clone https://github.com/seyioyemade/book-store.git
- cd book-store
- npm start


### Install

Install this project with:

- npm Install
